MyFlix API Documentation

This application will allow users to search for information about all things movie related. This includes info about actors, directors and genres. In addition, it will allow them to set up an account to create a list of their favorite movies.

Business Logic URL HTTP method Request body data format Response body data format
Get list of all movies. /movies GET None JSON object holding data about all movies. Example:

{ name: "Star Wars: The Empire Strikes Back", description: "After the Rebels are brutally overpowered by the Empire on the ice planet Hoth, Luke Skywalker begins Jedi training with Yoda, while his friends are pursued by Darth Vader.", genre: "sci-fi", director: "Irvin Kershner", img: "https://m.media-amazon.com/images/M/MV5BYmU1NDRjNDgtMzhiMi00NjZmLTg5NGItZDNiZjU5NTU4OTE0XkEyXkFqcGdeQXVyNzkwMjQ5NzM@._V1_SX300.jpg", featured: "false" }

Get data of single movie by title to user. /movies/[title] GET None JSON object holding data about a single movie.
Get data about a genre. /genre/[name] GET None JSON object holding data about a movie genre, containing name and description. Exampl:

{ Name: "Sci-fi", Description: "A movie about space and aliens." }

Get data about a director by name. /director/[name] GET None JSON object holding data about a director. Example:

{ Name: "George Lucas", Bio: "Science fiction director", Birth: 1944 }

Add a new user to registry. /users POST JSON object holding data about the user being added. Example:

{ username: "Jim Smith", password: "s3cr3tpassword", email: "him.smith@msn.com", DOB: "2/5/1955" }

JSON object holding data about the user that was added. Example:

{ username: "Jim Smith", password: "s3cr3tpassword", email: "him.smith@msn.com", DOB: "2/5/1955" }

Update a user's information. /users/[username] PUT JSON object holding data about the user being added. Example:

{ username: "Jim Smith", password: "s3cr3tpassword", email: "him.smith@msn.com", DOB: "2/5/1955" }

A message confirming a user's information was updated.
Add a movie to the user's favorites list /favorites/[username]/movies/[movieID] POST None A message confirming a title was added to the user's list of favorite movies.
Delete a movie from the user's list of favorites. /favorites/[username]/movies/[movieID] DELETE None A message confirming a title was deleted from the user's list of favorite movies.
Remove a user from the registry. /users/[username] DELETE None A message confirming the user was removed from the registry.
